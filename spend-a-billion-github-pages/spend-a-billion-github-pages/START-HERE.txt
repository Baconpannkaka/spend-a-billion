SPEND A BILLION – PUBLICERA EN TESTSIDA PÅ GITHUB

Du behöver inte installera något program.

1. Skapa ett nytt PUBLIC repository på github.com, exempelvis "spend-a-billion".
2. Välj Add file -> Upload files.
3. Öppna den här mappen och dra in ALLA filer och mappar till GitHub.
   Viktigt: .github-mappen måste följa med.
4. Välj Commit changes.
5. Gå till Settings -> Pages och välj GitHub Actions under Source.

Öppna sedan fliken Actions. När "Build and deploy Spend a Billion" är grön
finns länken till testsidan under Settings -> Pages.

Den fullständiga guiden finns i README.md.
